# GodBot
> A discord bot, capable of playing music, moderation and much more.

## Built With
- [_ASP.NET Core 6.0_](https://docs.microsoft.com/en-us/aspnet/core/release-notes/aspnetcore-6.0?view=aspnetcore-6.0)
- [_Discord.Net_](https://github.com/discord-net/)

## Setup
1. To get started, clone the project.
```
git clone https://github.com/WaadSulaiman/GodBot.git
```
2. Change the settings inside `AppSettings.cs`.
3. Navigate to `GodBot.Executable/Lavalink` run this java command in the terminal.
```
java -jar Lavalink.jar
```
4. Run.
